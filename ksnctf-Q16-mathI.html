公開鍵のeを使って、c = me mod nの計算式で暗号化されてるので、
秘密鍵のdで、m = cd mod nを計算して、FLAGを取り出せばよいらしい。

dはどっからきたの

拡張ユークリッド互除法で求めるらしい。
ex + ly = 1となる、xとyの組み合わせを求めて、xがd（秘密鍵）となるらしい。

  • e = 公開鍵
  • l = p-1とq-1の最小公倍数

答え

import math
import codecs

def gcd2(a, b):
    """拡張ユークリッド互除法"""
    if b == 0:
        u = 1
        v = 0
    else:
        q = a // b
        r = a % b
        (u0, v0) = gcd2(b, r)
        u = v0
        v = u0 - q * v0
    return (u, v)
